# 📞 ON:AIR

### 연결이 서툰 당신에게, 첫 신호를 함께합니다

**ON:AIR**는 전화 통화에 대한 불안을 느끼는 사용자를 위해<br>
AI와 함께 실제 통화와 유사한 환경에서 말하기 연습을 제공하고, <br>
통화 후 피드백을 통해 말하기 자신감을 회복하도록 돕는 **AI 기반 말하기 연습 플랫폼**입니다.

---

## 🔍 서비스 배경

전화 통화는 여전히 예약, 상담, 면접 등 다양한 상황에서 요구되지만
메신저와 텍스트 중심의 소통에 익숙한 사용자일수록
전화 통화에 심리적 부담을 느끼는 경우가 많습니다.

기존의 말하기 교육이나 상담 서비스는
실제 통화 환경과의 차이, 반복 연습의 부담 등으로 인해
실질적인 연습 경험을 제공하는 데 한계가 있습니다.

---

## 🎯 서비스 목표

* 전화 통화에 대한 심리적 부담을 줄이는 연습 환경 제공
* 실제 상황과 유사한 대화를 통한 말하기 경험 축적
* 연습과 피드백을 반복하며 자연스럽게 자신감 형성

---

## 👤 대상 사용자

* 전화 연결 자체에 부담을 느끼는 사용자
* 말실수에 대한 두려움으로 통화를 피하는 사용자
* 실제 통화 전에 안전한 연습 공간이 필요한 사용자

---

## 💡 서비스 핵심 개념

ON:AIR는 AI를 통화 상대방으로 설정하여
실제 사람과의 통화에서 느낄 수 있는 부담을 줄이고,
상황별 시나리오와 즉각적인 피드백을 통해
자연스럽게 말하기 연습을 이어갈 수 있도록 설계되었습니다.

---

## 🧩 주요 기능

* **상황별 통화 시나리오**

  * 예약, 상담, 면접 등 실생활 기반 시나리오 제공
* **AI 기반 실시간 대화**

  * 음성 인식(STT)과 음성 출력(TTS)을 활용한 통화 경험
* **통화 후 피드백 제공**

  * 말의 속도, 표현 반복, 대화 흐름 분석
* **연습 기록 관리**

  * 통화 기록과 피드백 데이터 저장

---

## 🔄 사용 흐름

1. 회원가입 및 로그인
2. 연습 시나리오 선택
3. AI와 통화 진행
4. 통화 종료 후 피드백 확인
5. 개선 내용을 반영하여 재연습

---

## 🧱 시스템 구성

* **Frontend**

  * 사용자 인터페이스, 통화 제어 기능
* **Backend**

  * 통화 처리, AI API 연동, 데이터 관리
* **AI**

  * 대화 생성 및 음성 처리
* **Database**

  * 사용자 정보, 통화 기록, 피드백 데이터 저장

---

## 🛠 기술 스택

* **Frontend**: HTML, CSS, JavaScript, Figma
* **Backend**: Python (Flask)
* **AI**: Google Gemini API, STT / TTS
* **Database**: Firebase Auth, Firestore

## 🔁 개발 후기

* 실제 통화와 유사한 흐름을 유지하는 것이 사용자 몰입도에 중요하다는 점을 확인했습니다.
* AI를 활용한 피드백 제공이 반복 연습에 대한 부담을 낮추는 데 효과적이었습니다.
